Veacon portal is a web based management platform which allows users to create and deploy custom scenarios, manage Veacon sensors and analyze location data, all with click of a button.

GET STARTED OR CONTACT US

Veacon portal will help you...

BUILD CUSTOM SCENARIOS

Using our dashboard you can easily create custom scenarios without writing a single line of code. Just follow the step by step guide and begin creating your first Veacon scenario. Scenarios can include messages, URLs and custom actions.

MANAGE VEACON SENSORS

Sensors must be configured in order to communicate with the portal. Once configuration is complete, you can start managing sensors through our dashboard. You can create projects, edit tags or enable/disable scenarios from the portal.

ANALYZE BEHAVIOR

Analyze user behavior and track performance of location dependent sensors. Use metrics such as most used tags, top sensors, most active scenarios active/passive sensors and more to optimize your Veacon experience.

MANAGE PROJECTS

Innovation has no barriers, neither do we! You can create and manage unlimited number of projects. Veacon Portal allows you to easily create, manage and analyze all your iBeacon projects from a single dashboard.
